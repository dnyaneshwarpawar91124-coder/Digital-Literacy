Digital Literacy Project Overview

The Digital Literacy Project aims to develop awareness and essential skills needed to use digital platforms effectively in today’s world. As a Student Digital Ambassador, this project focuses on guiding fellow students toward safe, responsible, and professional online behavior.

The project is organized into five modules, each covering important areas such as understanding digital tools, building an online presence, using collaboration platforms, practising proper communication, and learning about cyber safety. All activities and outputs have been systematically documented in a GitHub repository along with personal reflections.

Module Summary

🔹 Module 1: Digital Literacy Awareness
Designed a one-page infographic using Canva to explain the concept of digital literacy, its importance, and key internet safety practices.

🔹 Module 2: Student Digital Portfolio
Created and managed professional profiles on platforms like GitHub and LinkedIn to establish an online identity and highlight academic progress.

🔹 Module 3: Coding & Collaboration Platforms
Explored platforms such as HackerRank and developed a Google Form quiz to learn about teamwork, data collection, and analysis tools.

🔹 Module 4: Email Etiquette & Communication
Practised writing formal emails and developed a checklist of Do’s and Don’ts for responsible social media usage.

🔹 Module 5: Cybercrime Awareness
Studied cybercrime-related topics and prepared a checklist of preventive measures to encourage safer online practices among students.
